# Go 语言

## 什么是 Go 语言

> 2009年由Google公司开发出的开源编程语言

## 为什么使用 Go 语言

- 脚本语言开发速度
- C/C++运行速度
- 支持并发编程
- 为大数据、微服务，并发而生的通用编程语言

## Go feathers

- 简洁、快速(开发效率和运行速度)、安全
- 并行(并发处理能力)、有趣、开源
- 内存管理、数组安全、编译迅速

## Go 目前的应用

- Docker
- Kubernet
- Caddy
- Codis(豆瓣)
- Glow(类似于 Hadoop)
- Cockroach(蟑螂) 数据库
